Create a set of conditions that define who can take your study. Make sure that all the conditions that are listed in an “All” must be true and that at least one of the conditions listed in an “Any” must be true. Use “End all” and “And any” to close a list of conditions. The gray box below shows the condition in plain words. A red “Rule is not ready” will appear whenever the rules are not ready to use (e.g., if you haven’t closed an “Any” list with “End any”). You can save the file that was created into your local computer or load a pre-existing file from your computer and edit it with the generator. Once finished, press the 'Done' button and the rule will be saved to the study folder on the server.    Learn more
